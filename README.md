Temat: System Zarządzania Recepcją Siłowni - OLIMP Projekt nowoczesnego systemu recepcji siłowni.

System umożliwia rejestrację klientów, zarządzanie karnetami oraz obsługę grafiku zajęć.

Jak uruchomić projekt lokalnie?

1.  Należy otworzyć folder interface_olimp w programie Visual Studio Code.

Upewnij się, że masz zainstalowane rozszerzenie Live Server.

Kliknij przycisk "Go Live" w prawym dolnym rogu paska stanu VS Code.

Strona powinna automatycznie otworzyć się w przeglądarce.

lub

2.  Otwórz terminal (wiersz poleceń) w głównym folderze projektu.

Zainstaluj wymagane paczki i zależności komendą:

npm install

Następnie uruchom serwer deweloperski:

npm run dev W terminalu pojawi się adres lokalny (np. http://localhost). Skopiuj go i wklej do przeglądarki.
